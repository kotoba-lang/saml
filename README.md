# saml

SAML 2.0 substrate: AuthnRequest and Assertion verification result shapes.
XML signature verification, metadata fetch, and clock policy are host ports.

For the raw OASIS SAML 2.0 spec-as-data substrate (zero deps, no host
assumptions), see [kotoba-lang/org-oasis-saml](https://github.com/kotoba-lang/org-oasis-saml).
This repo is the result-shape/substrate layer consumed by
[kotoba-lang/authentication](https://github.com/kotoba-lang/authentication).
